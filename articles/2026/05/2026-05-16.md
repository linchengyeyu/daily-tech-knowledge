## 到底是个什么东西？

用最简单的话说：一个网页版的3D模型工作室。

你上传一张照片——产品、建筑、手办、动物，什么都可以。

AI分析图片内容，然后自动生成一个3D模型。

不是那种假的3D（就是照片转个角度糊弄人的），是真的可以360度旋转查看的立体模型。

可以放大看细节，可以缩放，可以截图，还可以导出成标准3D文件。

导出的GLB文件能干嘛？导入PPT做展示、导入视频软件做动画、甚至直接发给3D打印店打印出来。

我打个比方：以前做一个3D模型，要么花几百块请设计师，要么自己学Blender学一个月。现在拍照就行。

项目体积149MB，README写了7700多字，支持6种不同的3D生成引擎——看起来不是那种"几天就烂尾"的玩具项目。

## 三个让我觉得有意思的地方

![核心亮点](http://mmbiz.qpic.cn/mmbiz_png/5aIcUh1sVkZYkjv0Q4pBtYKcJ9JxGGMfBCGtWATqsAp3ltquxdNfyibfkwkneXriaN506ID56ebebb4CF843KlnbpRibDpSjZ0Hyfly0RBEqSo/0?wx_fmt=png)

### 第一个：6种AI引擎随便选

背后接了6个3D生成服务：Hyper3D、Tripo、Fal.ai、腾讯混元3D、JS Depth、还有本地GLB导入。

什么意思呢？同一个照片，你可以用不同的AI引擎各跑一次，对比效果，选最好的。

就像翻译软件一样——同一个句子扔给Google翻译和DeepL，哪个翻译得好用哪个。

我查了一下各家的价格。Tripo免费账号每月能生成10个模型，够你试试水。Hyper3D和Fal.ai按次收费，大概几毛钱到一两块钱一个模型。腾讯混元3D目前可以免费申请额度。

说实话，比起以前花几百块请人做一个模型，这个价格基本可以忽略。

你觉得这6个引擎哪个效果最好？评论区聊聊，我很好奇。

### 第二个：专业级的三栏工作台

界面长这样：左边是模型库，中间是3D舞台，右边是工具箱。

拖拽旋转、滚轮缩放、隔离查看零件、检查材质、截图、录视频——这些在Blender里才有的操作，它全塞进浏览器了。

还有一个"演示模式"，点一下自动隐藏所有面板，AI帮你规划镜头运动路径，录制一个干干净净的产品展示视频。

做电商的朋友应该懂这个价值——一个产品展示视频，外包做起码几百块。这个零成本。

### 第三个：不联网也能玩

如果你不想接API，或者暂时没申请到API Key，没关系。

直接把电脑上的 `.glb` 或 `.gltf` 3D文件拖进去，一样可以旋转查看、截图录视频。

所以它不只是"照片变3D"，还是一个挺好用的3D文件查看器。

就算你完全不用AI生成功能，光拿来查看3D文件也值了。

## 泼点冷水

说了这么多好的，说说限制。

这也是我一直强调的——推荐东西不说坑，跟骗人没区别。

![泼冷水](http://mmbiz.qpic.cn/mmbiz_png/5aIcUh1sVkYdMe7bw7wL8LDvBvibQicVnjMwiaT2SiaHQasGE3ViaExharbfmpPMxXNHdgNz2BGxAbcWbaOggvvcd0z5jHUPuU7Z7jToIFU7cHho/0?wx_fmt=png)

**限制一：照片变3D要花钱。**

核心功能需要第三方AI接口，大部分按次收费。Tripo免费账号每月10次，Hyper3D和Fal.ai按次收，腾讯混元3D可以白嫖。不接API的话，只能导入现成的3D文件，生成不了新的。

**限制二：一个人做的，才5天。**

作者huangserva，GitHub上还有十几个其他AI工具项目。其中skill-prompt-generator有1300多Star，claude-code-cli有580多Star。

说明是个有经验的开发者，不是新手随便玩玩。但毕竟一个人维护，可持续性说不好。万一作者不更新了，项目就停了。

Issues里有6个讨论——有人要加太阳系动画，有人要接Fal.ai，有人反馈大屏缩放有问题。说明用户活跃，但功能还在快速迭代中。

**限制三：3D生成质量离完美还远。**

简单物体（杯子、家具、方形产品）效果不错。

复杂物体（手指、透明材质、毛发）效果打折明显。

这是目前所有照片转3D技术的通病，不单是这个项目的问题。如果你需要的是珠宝级的精细3D模型，目前AI还做不到，还是得请设计师。

**限制四：需要会敲命令行。**

虽然是网页应用，但你得打开终端、敲 `git clone` 和 `npm install` 才能跑起来。

对程序员来说是两分钟的事。对完全没接触过代码的人，可能要查半天教程。

不过比起学Blender，这个门槛已经低得不能再低了。

## 怎么跑起来？

![使用方法](http://mmbiz.qpic.cn/mmbiz_png/5aIcUh1sVkaqibXT1z37sZwIB7lDMXXAlEVDwzEdl0pO17QleMFzKianowPnJKKO3hhgENnmMQGX0EDRoUqxZMJxUzbgNhEicicxgiaxsODFkicO8/0?wx_fmt=png)

两步就行。

**第一步：克隆 + 安装**

```
git clone https://github.com/huangserva/3DCellForge.git
cd 3DCellForge
npm install
```

**第二步：启动**

```
npm run dev
```

打开终端里显示的地址（一般是 `http://localhost:5173`）。

想用AI生成3D？复制 `.env.local.example` 为 `.env.local`，填上API Key。

不想填也行，直接拖本地3D文件进来用。

项目地址：https://github.com/huangserva/3DCellForge

觉得有用的话去点个Star。
