# 75个AI设计模板免费用，一键发到微信公众号，这个项目5天2400星

前两天刷GitHub Trending，看到一个项目刚上线5天就2400多Star。

仔细一看，不是什么大厂出的，是一个叫nexu-io的团队做的。

这个团队之前做了个Open Design，40K Star，200多个贡献者——靠谱的团队。

这次他们做了个新东西叫**html-anything**。

简单说就是：让AI帮你直接写HTML，不是写代码那种，是做出你能直接发布的内容。

公众号文章、小红书卡片、PPT、海报、简历……75个模板，9种设计场景。

你可能会问，这跟Gamma、Canva有什么区别？

区别大了——Gamma和Canva是在线的，要注册要付费（Gamma Pro要$10/月，Canva Pro要$120/年），而且模板是固定的，改不了底层。

html-anything是开源免费的，所有模板都是HTML代码，你能看到每一行，想改就改。

它背后支持8种AI工具：Claude Code、Cursor、Codex、Gemini CLI、GitHub Copilot CLI、OpenCode、Qwen Coder、Aider。

装了哪个就用哪个，不用额外花一分钱。

说实话，这种"把AI能力直接塞进设计工具"的思路，比那些"AI帮你写文案"的工具实用多了。

75个模板听着多，但每个都有具体的名字和风格。

比如"guizang-editorial"做出来像电子杂志，"swiss-international"是瑞士国际主义风格，"xiaohongshu"就是专门做小红书卡片的。

不是那种通用模板套一套就完事的。

打个比方，就像你去餐厅点菜。

以前AI只能给你一份食材清单和菜谱（Markdown）。

现在html-anything直接帮你炒好端上桌（HTML）。

你打开就能看，复制就能发。

## 02 我觉得最猛的几个点

**第一，不用API Key。**

市面上大部分AI设计工具都得注册账号、绑定信用卡、买额度。

这个项目不需要任何API Key。

它直接复用你电脑上已经装好的AI编程工具。

Claude Code、Cursor、Codex、Gemini CLI、GitHub Copilot CLI、OpenCode、Qwen Coder、Aider——8个工具自动识别。

你装了哪个就用哪个，不用额外花一分钱。

我在想，这招挺聪明的。现在用AI编程工具的人越来越多，与其再做一个AI服务，不如直接借用你已经有的。

**第二，9种设计场景全覆盖。**

说实话，这9个场景覆盖了大部分做内容的需求。

就拿小红书来说，做自媒体的人都知道小红书最费时间的就是做图。

做自媒体的朋友都知道，同样的内容要发到不同平台，排版是最烦的。

公众号要处理图片大小，小红书要做竖版图，知乎的格式又不一样。

三步就行：

1. 确保电脑上装了Claude Code或者Cursor之类的AI编程工具（装一个就够了）
2. 克隆项目：`git clone https://github.com/nexu-io/html-anything`，然后`npm install`和`npm run dev`
3. 打开浏览器，从75个模板里挑一个，输入你的内容，AI帮你生成

说实话这个安装过程对零基础的人来说还是有点门槛。

但如果你本身就在用AI工具，那基本就是30秒的事。

技术栈是Next.js，跑起来就是一个本地网页，打开就能用。

## 05 我的看法

这个项目的核心理念其实挺超前的。

它想做的事情是：**既然你已经让AI帮你干活了，为什么不让AI直接输出成品？**

我们现在的AI工具链是：AI写Markdown → 你自己排版 → 发布。

html-anything想变成：AI直接写HTML → 你检查一下 → 发布。

少了中间那个最费时间的排版环节。

等哪天连AI编程工具的安装都简化了，那才是真正全民AI设计时代。

在那之前，这个项目对已经在用AI工具的人来说，是个不错的提效神器。

你平时做内容花最多时间的是哪个环节？评论区聊聊，说不定下期就测这个方向。
