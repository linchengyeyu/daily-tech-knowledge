这个项目叫 **3DCellForge**。

一个AI驱动的3D细胞生成工作室。

我拿大白话说：你给它一张细胞的照片，它给你生成一个可以360度旋转的3D模型。

就像你以前看生物课本，细胞结构是平面的，印刷在纸上。

现在呢？这个项目把那张平面图变成了一个可以拿在手里转来转去的立体模型。

## 01 项目简介

项目背后的技术栈不复杂：React + Three.js + Vite。

翻译成人话就是：一个网页应用，打开浏览器就能用，不需要装任何东西。

它支持两种方式生成3D模型。

**第一种：调用Tripo的云服务。** 你上传一张细胞图片，Tripo的AI帮你生成3D模型。

**第二种：用本地的Hunyuan3D。** 不需要联网，但需要你自己搭环境。

两种都不想用？也行。项目自带了几个示例模型，打开就能玩。

## 02 核心亮点

**亮点一：细胞器交互式查看**

你能看到细胞膜、细胞核、内质网，每个细胞器都标了名字。点击任意一个，会弹出详细信息卡片。还有个显微镜参考图，方便你对照着看。

**亮点二：电影院模式**

把3D模型渲染成高清大片，支持导出截图和GLB文件。GLB是什么？就是一种3D文件格式，相当于3D世界的"MP4"。导出之后，你可以把模型导入到其他3D软件里继续编辑。

**亮点三：对比面板**

可以同时看两个不同的细胞，找它们的区别。对学生物的同学来说，这个功能很实用。

## 03 说点大实话

说完了好的，说说限制。

**第一：不免费。** Tripo API是按调用次数收费的。生成一个3D模型就要花一次钱。学生党可能不太友好。

**第二：效果取决于你给的图片。** 图片质量差，生成的3D模型也好不到哪去。就像你给厨师一堆烂菜叶，再厉害的厨师也做不出米其林。

**第三：只有一个人在做。** 整个项目就一个贡献者。这意味着如果作者不更新了，这个项目可能就停了。GitHub上目前只有2个Issue，其中一个还是广告spam。社区活跃度不算高。

**还有个现实问题：133MB的仓库大小。** 因为里面包含了预生成的3D模型文件。你克隆这个仓库的时候，得等一会儿。

## 04 怎么用

安装就两步。

```bash
git clone https://github.com/huangserva/3DCellForge.git
cd 3DCellForge && npm install && npm run dev
```

打开浏览器访问它给你的地址，就能看到效果了。

想用AI生成自己的3D细胞模型？需要先申请一个Tripo的API Key，然后创建 `.env.local` 文件，把Key填进去。

项目地址：https://github.com/huangserva/3DCellForge

记得去点个Star。
