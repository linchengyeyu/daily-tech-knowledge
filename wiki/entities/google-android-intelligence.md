# Google Android Intelligence

Google 推出的 Android 系统 AI 智能功能，支持跨应用多步自动化和 AI 增强鼠标指针。

## 基本信息

| 项目 | 详情 |

| 名称 | Google Android Intelligence |
| 平台 | Android |
| 发布方 | Google |
| 报道日期 | 2026-05-13（AIHOT速报） |

## 核心功能

### 跨应用多步自动化

- **跨应用操作**：AI Agent 可以跨越多个应用执行多步骤任务链
- **自然语言驱动**：用户用自然语言描述意图，AI 自动拆解为多步操作并执行
- **Agent 范式落地**：将 [[small-model-orchestration|AI Agent]] 范式从云端/桌面延伸到移动端

### AI 增强鼠标指针

- **智能指针**：鼠标指针集成 AI 能力，可根据上下文智能预判用户意图
- **交互革新**：传统指针从被动定位工具升级为主动辅助工具

## 技术意义

- **移动端 Agent 里程碑**：Android 占全球移动 OS 最大份额，跨应用自动化意味着 AI Agent 触达数十亿用户
- **与 [[generic-agent]] 等开源 Agent 框架形成对比**：Google 将 Agent 能力做进系统层，而非第三方工具
- **隐私考量**：跨应用操作意味着 AI 需要访问多应用数据，隐私边界需关注
