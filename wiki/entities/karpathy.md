# Andrej Karpathy

AI 研究者，前 Tesla AI 总监、OpenAI 联合创始人。个人知识管理领域提出了 LLM Wiki 模式。

## 贡献

- **LLM Wiki**：提出用 LLM 维护结构化 Wiki 的知识管理方法（发布为 GitHub Gist）
- 核心思想：LLM 是程序员，Wiki 是代码库，知识持续编译而非每次临时检索
- 推荐使用 Obsidian Web Clipper 收集素材
