# Matt Pocock Skills

TypeScript 大佬 Matt Pocock 出品的 Claude Code Skills 合集。口号：Skills for Real Engineers。

## 针对 AI 编程 4 个失败模式

| 失败模式 | 对症技能 |
|----------|----------|
| 需求对不齐 | /grill-me、/grill-with-docs 深度问答 |
| AI 输出啰嗦 | 共享语言 + CONTEXT.md |
| 代码容易出错 | /tdd 强制测试驱动开发 |
| 架构变面条 | /caveman、/zoom-out、/improve-codebase-architecture |

## 其他技能

/diagnose 排查问题、/triage 分类 issue、/to-issues 拆任务、/to-prd 写产品文档

## 安装

```bash
npx skills@latest add mattpocock/skills
```
