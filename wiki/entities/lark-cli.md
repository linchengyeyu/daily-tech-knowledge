# Lark CLI

飞书开源命令行工具，允许 AI 通过命令行直接操作飞书。

## 核心特点

- **AI 原生操作**：AI 可通过命令行直接操作飞书（建群、建文档、发通知等）
- **每步可预览、可审查**：操作透明，与 MCP 模式形成对比（可见可控 vs 云端黑盒）
- **开源生态**：外部已有 10 位开发者贡献代码被合并，钉钉和企业微信为零

## 里程碑

- **45 天 GitHub 破万星**：国内首个破万星的办公套件开源项目

## 与 MCP 模式对比

| 维度 | Lark CLI | MCP 模式 |
|------|----------|----------|
| 可见性 | 每步操作可预览、可审查 | 云端黑盒 |
| 可控性 | 命令行直接操作 | 依赖云端服务 |
| 开放性 | 开源，接受外部贡献 | 通常封闭 |
